# 🌻 Dilo — Your AI Emotional Support Companion
> !!!! This is a demo version. The project is still actively being developed.

Dilo is an AI-powered mental wellness chat app designed to make emotional support feel personal, warm, and accessible. Whether you need a friendly conversation or a more reflective, psychologist-style interaction — Dilo adapts to you.

## Features

- **Two AI Modes** — Switch between Friendly Mode and Psychologist Mode anytime
- **Mood-Based Recommendations** — Select how you're feeling and get personalized activity, food, and wellness suggestions
- **Multilingual Support** — English, Turkish, Arabic, Persian, Spanish, French, German (with proper RTL handling for Arabic & Persian)
- **Voice Input** — Speak your thoughts directly into the chat
- **Dark / Light Mode** — Comfortable for any time of day
- **Settings Panel** — Font size, theme, notification sounds, and language all in one place

##  Built With

- React + TypeScript
- Vite
- Google Gemini API
- CSS (custom, no UI library)

##  Run Locally

1. Clone the repo
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env.local` file and add your Gemini API key:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```
4. Start the app:
   ```bash
   npm run dev
   ```
5. Open `http://localhost:5173`

Get your free Gemini API key at [aistudio.google.com](https://aistudio.google.com)

## Vision

Mental health support should not feel clinical or out of reach. Dilo exists to bridge that gap — a small, thoughtful space where anyone can feel heard.

## Inspiration

Dilo's name and soul come from my cousin **Dilvin** — a psychology student who has always been there for me on my hardest days. Her empathy, patience, and genuine care for people inspired this entire project. I wanted to build something that carries even a small piece of what she naturally gives to everyone around her.
